Estudo Sobre a Linguagem Java

Linguagem java é uma linguagem de programação orientada a objetos, amplamente utilizada em aplicações empresariais e desenvolvimento web.

O que é esse Paradigma Orientada a objetos?

O paradigma de programação orientada a objetos (POO) é um modelo de programação que utiliza "objetos" para representar dados e métodos. Em POO, os objetos são instâncias de classes, que definem as propriedades e comportamentos dos objetos. Os principais conceitos da POO incluem encapsulamento, herança, polimorfismo e abstração.

O paradigma representa muito bem o domínio do problema, pois permite modelar entidades reais do mundo como objetos, facilitando a manutenção e expansão do código.

O que é abstração?

Abstração é o processo de ocultar detalhes complexos e mostrar apenas informações essenciais ao usuário. Em programação orientada a objetos, a abstração permite criar classes e métodos que representam conceitos do mundo real de forma simplificada, tornando o código mais legível e reutilizável.

Criação de uma classe Conta, que representa um Domínio de uma Conta bancária.

    
public class Conta {

    private String titular;
    private double saldo;
    private int numero;

    public Conta(String titular, int numero) {
        this.titular = titular;
        this.numero = numero;
        this.saldo = 0.0;
    }

    public void depositar(double valor) {
        if (valor > 0) {
            saldo += valor;
        }
    }

    public void sacar(double valor) {
        if (valor > 0 && valor <= saldo) {
            saldo -= valor;
        }
    }

    public String getTitular() {
        return titular;
    }

    public double getSaldo() {
        return saldo;
    }

    public int getNumero() {
        return numero;
    }
}

Os atributos:

Os atributos são as propriedades ou características do objeto. Na classe Conta, os atributos são:

  • titular: representa o nome do titular da conta.
  • saldo: representa o saldo atual da conta.
  • numero: representa o número da conta.
    
    private String titular;
    private double saldo;
    private int numero;

Os Modificadores de acesso:

Os modificadores de acesso determinam o nível de visibilidade dos atributos e métodos de uma classe. Os principais modificadores são:

  • private: o atributo ou método só pode ser acessado dentro da própria classe.
  • protected: o atributo ou método só pode ser acessado dentro da própria classe e por subclasses.
  • public: o atributo ou método pode ser acessado de qualquer lugar.

No exemplo da classe Conta, os atributos são declarados como private para garantir o encapsulamento e proteger os dados da conta.

Desta forma, ao definir os atributos como private, apenas utilizando os métodos públicos (getters e setters) da classe que posso acessar essas variáveis.

O atributo pode pertencer a classe, precisamos definir como static o seu tipo de Retorno:

  • static: o atributo pertence à classe e não a instâncias individuais.
            
        static int contador;  

        Conta(int numero){
        this.numero = numero;
        Conta.contador++;
     }
  • Não importa quantos objetos sejam criados, o valor do atributo static é compartilhado entre todos os objetos da classe.
  • Ao criar os objetos, automaticamente será acrescentado essa variável.

forma correta de acesso: 

            
        Conta.contador++;

Encapsulamento:

É utilizado nos atributos com o modificador de acesso private, onde eles só podem ser acessados pelos métodos que vão recuperar. Jababen termo utilizado para os métodos sets e gets.

A palavra chave "this":

A palavra-chave "this" é usada para referenciar o objeto atual dentro de um método ou construtor. Ela permite acessar os atributos e métodos do objeto instanciado.

O construtor da classe:

  • cada Classe possui um por padrão;
  • Não tem Retorno;
  • É chamado automaticamente quando um objeto é criado;
  • Inicializa os atributos do objeto.
        
        public Conta(String titular, int numero) {
        this.titular = titular;
        this.numero = numero;
        this.saldo = 0.0;
